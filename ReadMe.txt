-To compile and run the files
	input "make paint" for part B of the assignment
	input "make conway" for part c of the assignment

-Paint program
	1. When the program starts, it is initialized with random color, but not a shape type.
	2. Can change color, change shape, clear the board and quit whenever you want.
	3. Fist vertex of the line, rectangle and circle(center) are drawn for a better painting experience.

-Conway game
	1. Click and make your own cells then start the game of life. If you want to add or delete cell after the game started, you need to pause and click.
	2. Random starts the game with random cell, it starts directly.
	3. Timer is set to slow down or speed up two times as it was before.
